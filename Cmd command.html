Windows Command Line Reference

System File Repair

  • sfc /scannow: Scans and repairs corrupted Windows system files.

Disk Checking

  • chkdsk /f (Caution: Back up data first): Checks and fixes disk errors.
  • chkdsk: Checks for disk errors (read-only).

System Image Repair

  • DISM/Online /Cleanup-Image /CheckHealth: Checks the integrity of Windows system files.

System Tools

  • sysdm.cpl: Opens System Properties for configuration.
  • ncpa.cpl: Opens Network Connections for managing networks.
  • msinfo32: Shows detailed system information.
  • winget upgrade --all (Win10 v1809+): Upgrades all apps installed via winget package manager.
  • slmgr /rearm (Use with discretion: Licensing terms apply): Resets Windows activation timer (for major hardware changes).
  • View Saved Wi-Fi Network Names and Passwords:
  • netsh wlan show profiles: Lists all saved Wi-Fi network profiles.
  • netsh wlan show profile name="WiFiNetworkName" key=clear: Shows password for a specific Wi-Fi network.

Disk Cleanup

  • cleanmgr: Frees up disk space by deleting temporary files.

Other

  • netplwiz: Manages user accounts on your system.
  • wsreset: Resets the Windows Store cache (for troubleshooting Store issues).

Diagnostics (Use with Caution)

  • dxdiag: Provides information about graphics, sound, and input devices.
  • memtest86.com (External): Tests your computer's memory for errors (use cautiously).

External Tools (Use at Your Own Risk)

  • deadpixeltest.org (Website): Helps identify dead pixels on your monitor.
  • keyboardtester.com (Website): Tests your keyboard for proper functionality.
  • furmark (External): Performs a demanding graphics stress test (can damage hardware if not careful).

Network Troubleshooting

  • tracert google.com: Traces the route packets take to reach a website.
  • nslookup www.google.com: Looks up a website's IP address.
